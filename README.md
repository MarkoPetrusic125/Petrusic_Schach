# Petrusic_Schach
A chess game written in C# with basic rules and gameplay.

# Schachspiel in C#

Dieses Projekt ist ein Schachspiel, das in C# entwickelt wird. Der Fokus liegt auf sauberer Struktur, guter Testbarkeit und schrittweiser Entwicklung.

## Projektziel
Das Ziel dieses Projekts ist es, ein funktionierendes Schachspiel zu implementieren und dabei wichtige Programmierkonzepte wie objektorientierte Programmierung (OOP) und saubere Softwarearchitektur zu üben.

## Anforderungen
- Entwicklung in kleinen, überschaubaren Schritten
- Regelmäßiges Testen der Funktionalität
- Saubere Trennung von Logik und Benutzeroberfläche
- Keine Konsolenein- oder -ausgaben in der Bibliothek

## Testbarkeit
- Es wird stets ein Hauptprogramm bereitgestellt, um das Spiel zu testen
- Die Spiellogik befindet sich in einer separaten Bibliothek
- Tests und Ausführung erfolgen ausschließlich über das Hauptprogramm

## Projektstruktur
- Bibliothek: Enthält die gesamte Spiellogik (keine Console I/O)
- Hauptprogramm: Dient zum Testen und Ausprobieren des Spiels

## Entwicklung
Dieses Projekt wird kontinuierlich weiterentwickelt:
- Regelmäßige Commits
- Nachvollziehbare Entwicklungsschritte
- Verbesserungen und Erweiterungen werden laufend hinzugefügt

## Lernziel
- Verständnis für saubere Code-Struktur
- Umsetzung von Spielregeln in Code
- Arbeiten mit Git und GitHub
- Nachvollziehbare Versionskontrolle
